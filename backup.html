Setup Guide / උපදෙස් මාලාව

Google Drive එකට sync වීම සඳහා පහත පියවර අනුව Apps Script project එකක් deploy කරන්න:

  1. 1. Apps Script project එකක් හදන්න script.google.com වෙත ගොස් "New Project" ක්ලික් කරන්න.
  2. 2. Script code paste කරන්න පහත button එකෙන් script copy කර Apps Script Editor එකේ ඇති Default code ඉවත් කර paste කරන්න.
  3. 3. Web App ලෙස Deploy කරන්න Deploy → New Deployment → Web App. Execute as: Me (ඔබගේ Google Account). Who has access: Anyone ලෙස Deploy කරන්න. Generate වූ Web App URL ඉහළ field එකේ paste කරන්න.
